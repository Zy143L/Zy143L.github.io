使用说明:

此系统仅适用于GT106主板 (抖音9.9元充电宝随身WiFi)
使用adb命令进入9008模式 adb reboot edl
使用 Qualcomm Premium Tool 擦除设备的aboot boot分区
将压缩包内的 aboot.bin 刷入aboot分区

如果你拆开设备了 请断开电池 裸板刷机 做完上一步操作
直接拔电再插上 会出现BootLoader设备 执行flash.bat即可

如果你没有拆开设备 请捅RST孔+电源同时摁着 直到设备重启
出现BootLoader设备 执行flash.bat即可

基带工作正常 默认无需推送modem文件 即可使用外置卡
查询命令 mmcli -m 0

本固件无自带WiFi热点 开机插入电脑后模式为 ADB+RNDIS
(Win11下ADB概率不可用 请使用RNDIS配置)

固件默认用户名root 密码 admin

按键逻辑

  • 侧面小孔RST 为重启按键 点一下系统重启(复位重启)
  • 电源键 单击为IP5036激活(电池灯亮) 关机状态下 长按开机 开机后 双击按键关机(IP5306停止输出)

Bootloder模式 关机状态 按着RST 再长按电源开机 进入Bootloder模式
9008模式 进入Bootloder模式 输入指令fastboot oem reboot-edl进入

LED
红色为系统灯 WIFI绿色为网络灯 可自行配置

系统语言 ZH_CN UTF-8 使用USTC源
