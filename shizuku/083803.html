设备刷机需要的软件

发表于2022-08-11|更新于2022-08-13
|字数总计:1.1k|阅读时长:4分钟

刷机与制作需要的软件

电脑用户请参考侧面导航快速定位或借助Ctrl-F搜索

手机用户请点击右下方的目录按钮进行快速阅览

miko service tool

安装后 替换 loder.exe 文件作为主程序 请自行替换桌面快捷方式

功能

在9008模式下 进行刷入与读取 支持eMMC全分区备份

拿到UFI的第一件事就是备份eMMC全备份

自带大量引导文件 自动识别端口

其中的小米华为三星功能模块 请勿使用 !

下载地址

https://www.123pan.com/s/mh49-qyfwv 提取码:miko

Qualcomm Premium Tool

功能

自带firehose引导 自动重启端口 支持单分区的擦写读

安装后需要激活 安装包内自带注册机

请勿点击 9008 to 9006 选项

下载地址

https://www.123pan.com/s/mh49-6yfwv 提取码: qpts

QualTek Tool

功能

支持分区读取与擦写 写分区需要 rawprogram0.xml 文件

不能自动识别平台 需要手动选择Loader文件

下载地址

https://www.123pan.com/s/mh49-yyfwv 提取码: qual

星海工具

功能

修改IMEI MEID 强开9091端口 备份QCN 刷入QCN

其他功能因程序适配问题 请勿随意对手机尝试使用 !

下载地址

https://www.123pan.com/s/mh49-Vyfwv 提取码: svip

高通串号工具

功能

支持对高通平台设备修改SN MEID IMEI WIFI BT 识别码

缺点: 需要手动开启9091端口 建议配合星海工具

请勿在新手机上使用此工具 会导致校验失败导致不开机

下载地址

https://www.123pan.com/s/mh49-IVfwv

AndroidCarrierClassTools

功能

非常强大的ROM解包打包移植工具 也叫航母工具

适合用于解包Boot移植System等

支持ext4任意分区解包提取 功能多多

下载地址

https://www.123pan.com/s/mh49-jyfwv 提取码: acct

Microsoft VC运行库 / .NET运行库

功能

补全电脑缺失的运行库环境 很多程序依赖这些运行库 例如星海或者Miko

没有运行库软件会爆异常 请务必安装

下载地址

https://www.123pan.com/s/mh49-TVfwv

ADB_BootLoder 一键环境/驱动

来源于安智论坛 版本虽老 但是非常好用 Win10/11 测试一切正常

功能

15秒安装ADB_FastBoot环境与驱动 刷机必备

下载地址

https://www.123pan.com/s/mh49-vVfwv

MiFlash 单文件 x86/x64 版本

单文件版本 32位与64位双版本

功能

MiFlash 刷机神器 支持FASTBOOT模式与9008模式快速刷机

老版本支持旧系统 新版本对旧系统兼容性不佳 单文件版本方便使用

9008模式下 下方功能无任何作用

下载地址

https://www.123pan.com/s/mh49-dVfwv

MiFlash 2018版本

功能

MiFlash 刷机神器 支持FASTBOOT模式与9008模式快速刷机

2018版本支持新一些的系统刷机 无需登录小米账号 自带驱动安装

9008模式刷机导入会提示 couldn't find flash script.无需理会

下载地址

https://www.123pan.com/s/mh49-3Vfwv 提取码: 2018

SPRD IMEI修改工具

功能

展讯紫光芯片修改IMEI工具 需要开启设备的DIAG端口

需要自行选择DIAG端口号

搜狗输入法不兼容 点击设置闪退请切换输入法

下载地址

https://www.123pan.com/s/mh49-hVfwv

展讯紫光驱动

功能

包含32与64位双版本 SPRD设备必备驱动

下载地址

https://www.123pan.com/s/mh49-HVfwv

随身WiFi助手

功能

功能太多了..自己看图吧 入门必备 一键操作

下载地址

https://www.123pan.com/s/mh49-AVfwv

ARDC投屏工具

功能

支持安卓4.4投屏 (截图模式)

安卓5.0以上投屏 (虚拟设备模式)

下载地址

https://www.123pan.com/s/mh49-9Vfwv

Scrcpy投屏工具

功能

Scrcpy 开源地址 https://github.com/Genymobile/scrcpy

此版本为晨钟酱开发的ScrcpyGUI版本 使用简单 操作直观

支持安卓5.1以上设备投屏

下载地址

https://www.123pan.com/s/mh49-QVfwv

dtimg2dtb

开源地址 https://github.com/benjaminwan/dtimg2dtb_windows

功能

Boot.img解包后针对dtimg文件拆分为dtb设备树文件

用于系统移植设备树拆分

下载地址

https://www.123pan.com/s/mh49-EVfwv

APKDB 安卓逆手

功能

很方便的安卓APP逆向工具

下载地址

https://www.123pan.com/s/mh49-kVfwv

总结

以上软件均收集与互联网 如有侵权等问题 请与本人联系!
